HTML

  • HTML stands for Hypertext Markup Language.
  • HTML is one of the three main languages used for web development.
  • Elements are a type of content on a webpage, elements work together to build the structure of a webpage and provide a framework that makes it easy for a user to navigate a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is not only responsible for providing the structure of a website but also making sure that it provides a positive user experience.
  • The section element are semantic elements that define a section in a document that contains thematic content.

CSS

  • CSS stands for Cascading Style Sheets.
  • Using CSS on a webpage defines the colors, aesthetics, visual layout, and organization of the webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS is used to describe the presentation of a document written in HTML.

GIT

  • git status: checks what branch we are currenty on
  • git checkout -b branch name: creates a new branch and switches to it
  • Using Git and connecting it to our computer we created a way to save our work to a GitHub repo and make it easier to collaborate with others.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is a programming language that allows users to interact with websites.
